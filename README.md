# Bootstrap 4 Starter Pack 

Includes complete Bootstrap 4 dev environment with gulp and sass



### Version

1.0.0

## Install Dependencies

```bash
npm install 
```

## Compile Sass & Run Dev Server

```bash
npm start
```

Files are compiled into /src
